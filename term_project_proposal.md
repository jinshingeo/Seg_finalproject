# 텀프로젝트 제안 — 거주지 분리 측도 적용

## 1. 지역 선정

**서울시 동대문구** (14개 행정동)

- 경희대·한국외대 인접 → 외국인 유학생 밀집 지역
- 행정동 단위 국적별 외국인 데이터 확보 가능한 몇 안 되는 지역
- 동별 외국인 수 편차가 이미 뚜렷 (이문1동 3,686명 vs 답십리2동 382명)

---

## 2. 분리요소 선정

**국적별 거주지 분리** (Segregation by Nationality)

- 내국인 vs 외국인 (2집단) 또는 내국인 / 중국인 / 기타 외국인 (다집단) → **팀 논의 필요**
- 체류자격(D-2 유학) 동별 데이터 미제공 → 국적을 유학생의 proxy로 사용
  - 동대문구 외국인 다수가 외대·경희대 재학생임을 근거로 정당화 가능

측도는 OD 불필요한 것 중에서 선택 (팀 보유 R 패키지 기준):
- Isolation Index, Theil's H, Atkinson Index 등

## 4. 왜 이 데이터인가

- 국적별 × 동별 교차 데이터는 법무부가 **2016년까지만 생산** → 현재 공개된 가장 세밀한 공간 단위
- 행정동 단위는 측도 계산을 위한 spatial unit으로 바로 활용 가능
- 내국인 데이터와 동 이름 기준으로 합산 가능 → 추가 가공 최소화

---

## 5. 기대 인사이트

- 대학 인접 동(이문1동·회기동)에서 외국인 집중 → Isolation Index 높게 나오는지 검증
- 국적별로 분리 패턴이 다른지 (중국 vs 베트남 vs 기타)
- 전월세가 낮은 동에 외국인 집중되는지 → 경제적 배제 메커니즘 확인
- 집주인 차별(입주 거부) 관행과 분리 패턴 연결 — 질적 해석 근거 제시
