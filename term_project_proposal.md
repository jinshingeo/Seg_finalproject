# 텀프로젝트 제안 — 거주지 분리 측도 적용

## 1. 지역 선정

**서울시 동대문구** (14개 행정동)

- 경희대·한국외대 인접 → 외국인 유학생 밀집 지역
- 행정동 단위 국적별 외국인 데이터 확보 가능
- 동별 외국인 수 편차가 이미 뚜렷 (이문1동 3,686명 vs 답십리2동 382명)

---

## 2. 분리요소 선정

**국적별 거주지 분리** (Segregation by Nationality)

- 내국인 vs 외국인 (2집단) 또는 내국인 / 중국인 / 기타 외국인 (다집단) → **팀 논의 필요**
- 체류자격(D-2 유학) 동별 데이터 미제공 → 국적을 유학생의 proxy로 사용
  - 동대문구 외국인 다수가 외대·경희대 재학생임을 어떻게 보일지..? 논의 필요


---

## 3. 분석 구조 (2단계)

### 1단계 — 분리 측정
행정동별 국적 집단 간 거주지 분리 측도 계산

### 2단계 — 원인 설명: 왜 분리가 발생하는가?

두 가지 메커니즘이 복합적으로 작용한다고 가정:

| 메커니즘 | 내용 | 검증 방법 |
|----------|------|-----------|
| **수요측** | 고소득(=고집값) 내국인은 거주지 선택지가 넓어 외국인 없는 동을 선택 가능 | 동별 평균 월세 ↔ 외국인 비율 상관관계 |
| **공급측** | 집주인의 외국인 입주 거부 → 외국인이 저가 주거지 밀집 동에 몰릴 수밖에 없음 | 뉴스·사례 기반 질적 근거로 보완 |

> 집값(월세)은 소득의 직접 측정이 어려운 상황에서 사용하는 proxy.  
> 대학생 특성상 본인 소득보다 **부모의 주거 지원 수준**이 거주지를 결정하므로 집값이 더 적절한 변수.

## 4. 데이터

### 확보 완료
| 데이터 | 출처 | 비고 |
|--------|------|------|
| 국적별 × 행정동별 등록외국인 현황 | 법무부 / KOSIS | 2016년 (마지막 생산 연도) |
| 동대문구 연립다세대 전월세 실거래가 (월세금) | 국토교통부 | 2016년, 소득 proxy |

## 5. 왜 이 데이터인가

- 국적별 × 동별 교차 데이터는 법무부가 **2016년까지만 생산하는것으로 확인**
- 행정동 단위는 측도 계산을 위한 spatial unit으로 바로 활용 가능
- 전월세 실거래가는 소득 직접 데이터 부재 시 표준적으로 사용되는 proxy이며, 2016년 기준으로 시점 일치

---

## 6. 기대 인사이트

- 대학 인접 동(이문1동·회기동)에서 외국인 집중 → Isolation Index 높게 나오는지 검증
- 국적별로 분리 패턴이 다른지 (중국 vs 베트남 vs 기타)
- **전월세가 높은 동일수록 외국인 비율이 낮은지** → 수요측 경제적 분리 검증
- 전월세가 낮은 동에 외국인 집중 → 공급측 차별과 경제적 배제의 복합 결과로 해석
- 집주인 차별(입주 거부) 관행은 뉴스·커뮤니티 사례를 질적 근거로 제시
